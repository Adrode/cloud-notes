## Jakie są podstawy SQLAlchemy ORM?
#sqlalchemy #orm 
```python
from sqlalchemy import create_engine, Integer, String, Float, Column, ForeignKey
from sqlalchemy.orm import declarative_base, sessionmaker, relationship, Session

eninge = create_engine('postgres+psycopg2://postgres:postgres@localhost:5432/app_db', echo=True)

Base = declarative_base()

class Person(Base):
	__tablename__ = 'people' # nazwa tabeli w bazie
	id = Column(Integer, primary_key=True)
	name = Column(String, nullable=False)
	age = Column(Integer)
	# schemat tworzenia tabeli w ORM
	things = relationship('Thing', back_populates='people')
	
class Thing(Base):
	__tablename__ = 'things'
	id = Column(Integer, primary_key=True)
	description = Column(String, nullable=False)
	value = Column(Float)
	owner = Column(Integer, ForeignKey('people.id')) # ustawienie FK
	
	people = relationship('Person', back_populates='things')
	
Base.metadata.create_all(engine)
# podobnie jak w Core; utworzenie tabel, kolumn, PK, FK, itd.
...
```
- **`declarative_base()`** - klasa bazowa, po której dziedziczą moje kolejne klasy
- **`relationship()`** - Python nie wie o relacji po samym ForeignKey, więc zdefiniowanie `relationship()` jest dla Pythona
	- mogę wtedy zrobić `new_person.thing.description` i będę miał dostęp do tych danych z obiektu

```python
...
Session = sessionmaker(bind=engine)
session = Session()

new_person = Person(name='Sam', age=80)
session.add(new_person)
# dodanie rekordu do tabeli przez obiekty

session.execute(text('INSERT INTO people (name, age) VALUES ("Mike", 30);'))
# dodanie rekordu do tabeli przez SQL query

session.commit()

session.close() # zamyka sesję
```
- `Session` obsługuje transakcję
	- `Session = sessionmaker(bind=engine)` - to jest klasa
	- `session = Session()` - to jest obiekt / instancja
		- tworzy się to w taki sposób, aby móc tworzyć kolejne instancje session, które można potrzebować do innych transakcji

### Select, Update, Delete, Join, Group
```python
# SELECT
result = session.query(Person).all()
print([p.name for p in result])

# SELECT with filter
result = session.query(Person).filter(Person.age > 50).all()

# DELETE
result = session.query(Person).filter(Person.age > 50).delete()

# UPDATE
result = session.query(Person).filter(Person.age > 50).update({'name': 'Charles'})

# JOIN
result = session.query(Person.name, Thing.description).join(Thing).all()

# GROUP BY
result = session.query(Thing.owner, func.sum(Thing.value)).group_by(Thing.owner).having(func.sum(Thing.value) > 50).all()
```
