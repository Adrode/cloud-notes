#regular_expressions #regex
- **Regular expressions** - służą do ustalania / wyszukiwania patternów stringów
	- ```python
import re # import modułu do regexów

pattern = re.compile("pattern_regexu")

print(pattern.search("sprawdzany_string"))
# zwróci None, jeśli string nie zgadza się z patternem, lub zwróci obiekt re.Match, jeśli się zgadza
```
	- przykłady patternu regexów / znaczenia:
		- `^[A-Z]+$`:
			- `^` - tzw. *anchor*, czyli string musi zaczynać się od ustalonego typu znaków
				- czyli od czego pattern ma się zaczynać
			- `[A-Z]` - tylko wielkie znaki
			- `+` - tzw. *1+*, czyli minimum jeden znak lub więcej, spełniające poprzedzający warunek (tutaj wielkie znaki)
			- `$` - *anchor* oznaczający zakończenie patternu
				- czyli na czym pattern ma się kończyć
					- string `"HEllo World"` w metodzie search zwróci `None`
					- string `"HELLOWORLD"` zwróci re.Match, czyli dopasowanie
		- `^[a-zA-Z\s]+$`:
			- `[a-zA-Z]` - małe oraz wielkie litery
			- `[...\s]` - \s, czyli zezwolenie na spacje
				- string `"Hello WORlD"` zwróci dopasowanie
		-  `^[a-z]{3}[0-9]{3,5}[^a-zA-Z0-9]{1}[A-Z]{0,2}`:
			- pierwsze 3 znaki lowercase
			- 3-5 cyfr
			- 1 znak specjalny
				- `[^a-zA-Z0-9]` - tutaj '^' oznacza negację, czyli dopuszczalne jest wszystko inne poza lowercase, uppercase i digits
			- opcjonalnie do 2 wielkich znaków
				- string `"abc1945!AZ"` zwróci dopasowanie
		- `^.{10}$`:
			- `.` - oznacza dowolny znak
				- czyli tutaj będzie spełniał warunek dowolny string o długości 10 znaków
		- `^[a-zA-Z0-9\.\-_]+@{1}[a-zA-Z0-9]+\.{1}[a-zA-Z]{2,3}$`:
			- bardzo podstawowy pattern dla adresu email
