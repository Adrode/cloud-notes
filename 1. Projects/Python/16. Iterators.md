#iterators
- **Iterators** - obiekt, który zwraca elementy **one at a time** z sekwencji danych
	- obiekt w Python jest iteratorem jeśli ma
		- **`__iter__()`** - zwraca iterator własnego obiektu
		- **`__next()__`** - zwraca kolejny element w sekwencji
			- jeśli brak kolejnych elementów w sekwencji, to robimy `raise StopIteration`
```python
import random

class Dice:
  def __init__(self, rolls):
    self.rolls = rolls
    self.count = 0

  def __iter__(self):
    return self
  
  def __next__(self):
    if self.count < self.rolls:
      self.count += 1
      return random.randint(1, 6)
    else:
      raise StopIteration
    
dice = [die for die in Dice(3)]

print(dice)
```
