#records_modification
## Jak usunąć wiersz?
#DELETE
- **`DELETE FROM person WHERE id = '120';`** - usunie wiersz o wskazanym id; najlepiej wskazywać wiersze po PRIMARY KEY
	- **`DELETE FROM person;`** - usunie wszystkie wiersze w tabeli

## Jak zmienić wybraną wartość w wierszu?
#UPDATE
- **`UPDATE person SET email = 'awoz@gmail.com' WHERE id = 100;`** - zaktualizuje wskazaną kolumnę we wskazanym wierszu; bez WHERE byłby to update wszystkich wierszy
