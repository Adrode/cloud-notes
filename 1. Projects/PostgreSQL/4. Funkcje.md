## Jak działa COALESCE?
#COALESCE
- Działa jak **default value**, w przypadku gdyby nie było wartości w danej komórce
- **`SELECT COALESCE(price, 1) FROM car;`** - jeśli wartość w price będzie null (brak wartości), to wtedy wchodzi default value czyli 1
	- **`SELECT COALESCE(price_after_discount, original_price, 10000) FROM car;`** - można włożyć więcej kolumn / wartości do funkcji
	- **`SELECT COALESCE(email, 'Email not provided') FROM person;`** - inny przykład

## Jak działa NULLIF()?
#NULLIF 
- **NULLIF** - zwraca null jeśli obie wartości są równe, a jeśli są różne to zwraca pierwszą wartość
	- **`SELECT NULLIF(1, 1)`** - zwróci null
	- **`SELECT NULLIF(1, 2)`** - zwróci 1
	- **`SELECT COALESCE(NULLIF(1, 1), 0)`** - zwróci 0 jako default value od coalesce
