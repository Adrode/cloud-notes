## Do czego służy PRIMARY KEY?
#primary_key
- **Primary key** - klucz, czyli kolumna która jednoznacznie identyfikuje każdy wiersz. Może być tylko jeden klucz na tabelę.

## Jak usunąć PK z istniejącej tabeli?
#primary_key
- **`ALTER TABLE person DROP CONSTRAINT person_pkey;`**
	- po komendzie `\d table_name` można zobaczyć nazwę primary key, np. `person_pkey`

## Jak dodać PK do istniejącej tabeli?
#primary_key
- **`ALTER TABLE person ADD PRIMARY KEY (id)`** - doda PK do tabeli; jeśli któraś z wartości nie będzie unikalna, to dostaniemy błąd
